# A11y Scan

Agent supports `page.a11yScan` via @axe-core/playwright.

## Options

```
{
  include?: string[]
  exclude?: string[]
  tags?: string[]
  includedImpacts?: Array<'minor'|'moderate'|'serious'|'critical'>
  resultDetail?: 'summary'|'full'
}
```

## Result

```
{
  ok: boolean,
  url: string,
  ts: number,
  violations: [...],
  counts: { total, byImpact },
  evidence?: { screenshotPath },
  raw?: unknown
}
```

## Notes

- In `summary` mode (default), output is compact.
- When violations exist, screenshot is saved to `.artifacts/a11y/<ts>.png`.
