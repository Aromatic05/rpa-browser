# 无障碍 (A11y)

当前项目的自动化主要依赖 **Playwright accessibility snapshot**：

- `trace.page.snapshotA11y`：输出可访问性树（JSON 字符串）。
- `trace.locator.*`：基于 a11yNodeId 或 a11yHint（role/name/text）进行元素绑定。

## 说明

- step 执行优先依赖 snapshotA11y；当 `a11yHint` 解析失败时会返回结构化错误（如 `ERR_NOT_FOUND`）。
- 旧的 `page.a11yScan`/axe 扫描链路已移除，不再作为默认执行路径。
