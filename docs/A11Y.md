# 无障碍扫描 (A11y)

Agent 通过 `@axe-core/playwright` 支持 `page.a11yScan`。

## 选项

```
{
  include?: string[]
  exclude?: string[]
  tags?: string[]
  includedImpacts?: Array<'minor'|'moderate'|'serious'|'critical'>
  resultDetail?: 'summary'|'full'
}
```

## 结果

```
{
  ok: boolean,
  url: string,
  ts: number,
  violations: [...],
  counts: { total, byImpact },
  evidence?: { screenshotPath },
  raw?: unknown
}
```

## 说明

- 在 `summary` 模式（默认）下，输出为精简格式。
- 若存在违规项，会将截图保存到 `.artifacts/a11y/<ts>.png`。
